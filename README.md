# Component Practice

React 컴포넌트 구현력과 상태 관리 패턴을 작은 문제부터 복잡한 문제까지 순서대로 연습하는 저장소입니다.

## 목표

- 작은 state부터 배열 state까지 단계적으로 익히기
- 이벤트 핸들러와 조건부 렌더링에 익숙해지기
- 목록 렌더링, 선택 상태, 토글 패턴 반복하기
- 동작하는 코드를 읽기 좋은 코드로 리팩터링하기

## 난이도 순서

1. 좋아요 토글
2. 카운터
3. 드롭다운
4. 카테고리 선택 / 해제
5. 로그인 폼
6. 탭 메뉴
7. FAQ 아코디언
8. 투두 리스트

## 복습 규칙

1. 먼저 동작하게 만든다.
2. state로 둘 값이 맞는지 다시 본다.
3. 조건에 이름을 붙인다.
4. 계산 가능한 값은 state로 만들지 않는다.
5. 리팩터링 후 커밋한다.

## 실행

```powershell
cmd /c npm run dev
```
