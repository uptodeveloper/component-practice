# Component Practice

React 컴포넌트 구현력과 상태 관리 패턴을 반복 연습하는 저장소입니다.

## 연습 목표

- `useState`로 바뀌는 값 관리하기
- 이벤트 핸들러로 상태 변경하기
- 조건부 렌더링 익히기
- 리스트 렌더링과 선택 상태 다루기
- 동작하는 코드를 읽기 좋은 코드로 리팩터링하기

## 연습 목록

- 로그인 폼
- 카테고리 선택 / 해제
- FAQ 아코디언

## 복습 규칙

1. 먼저 동작하게 만든다.
2. state로 둘 값이 맞는지 다시 본다.
3. 조건에 이름을 붙인다.
4. 계산 가능한 값은 state로 만들지 않는다.
5. 리팩터링 후 커밋한다.

## 실행

```powershell
cmd /c npm run dev
```
